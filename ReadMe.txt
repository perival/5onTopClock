5onTop Clock
=====

"5onTop Clock" is a decimal time clock (10 hours in a day, 100 minutes in an hour).
This clock puts the 5 at the top of the clock, so 5.00 decimal time (which corresponds to noon in old time) has the hour hand pointing up to 5, and the minute hand pointing straight down to 0.
	

Requirements
The supplied Xcode project was created using Xcode v4.6 or later running on at least Mac OS X 10.5 or later.
